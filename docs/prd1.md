一、项目概述

1.1 项目背景
蓝山工作室是重庆邮电大学信息化办公室指导下的学生开发团队，曾开发出"We重邮"等全国知名的校园产品。为提升工作室的品牌形象，吸引优秀的大一新生加入，需要建立一个官方网站，作为对外宣传和人才招募的核心窗口。

1.2 项目目标

- 核心目标：面向大一新生，在宣传场景下，通过网站迅速了解工作室概况，被吸引并了解参加考核的途径。
- 业务目标：提升工作室在校园内的知名度和专业感，成为展示团队实力与技术文化的"名片"。
- 体验目标：打造一个加载迅速、视觉风格统一、多端适配良好的品牌官网。

1.3 项目范围

- 包含内容：首页、关于我们、组织架构、项目介绍、毕业去向、联系我们、加入我们等前台页面，以及配套的后台内容管理系统（CMS）。
- 不包含内容：在线考核系统、用户社区论坛、电商支付功能等。

---

二、用户与场景

- 目标用户：重庆邮电大学大一新生。他们是潜在的技术爱好者或对互联网产品感兴趣的学生，希望通过加入工作室提升实践能力。
- 使用场景：
  1. 在学长学姐的宣传推荐下，通过链接访问网站。
  2. 在新生群、朋友圈或社团招新材料中看到网站二维码，扫码访问。
  3. 主动搜索"蓝山工作室"以了解其背景和成果。

3.2 后台管理功能（CMS）【搁置】

后台管理需为非技术人员（如工作室运营成员）提供便捷的内容更新能力。

- 项目管理：支持新增、编辑、删除和上线/下线"项目介绍"模块中的项目条目。
- 毕业去向管理：支持新增、编辑、删除"毕业去向"模块中的成员信息。
- 组织架构管理：支持在后台对组织架构图或部门列表进行更新调整。
- 联系方式管理：支持修改"联系我们"及"加入我们"页面中的QQ群号、邮箱、报名链接等关键信息。
- 用户与权限管理：可设置不同角色的管理员账号，区分内容编辑、审核和超级管理员的权限。

四、设计需求

4.1 视觉风格

- 核心参照：以《明日方舟：终末地》官网风格为基调。
- 风格关键词：工业风、机能风、简洁、秩序感、高级感。追求视觉上的整体性与协调感，避免过度设计。
- 色彩方案：采用低饱和度配色作为主色调（如冷灰、深蓝灰），营造冷静、专业的科技感。使用明黄警戒色作为点缀和强调色，用于关键按钮、链接或高亮元素，强化工业氛围与引导性。
- UI元素：界面设计应具有工业仪表盘或数据面板的质感。可借鉴类似CMKY色条、坐标网格线等肌理作为装饰，体现系统性与逻辑性。
- 字体：HarmonyOS-sans(鸿蒙黑体)、Noto-sans（思源黑体）

4.2 交互与体验

- 多端适配：网站必须采用响应式设计，完美适配PC、平板和移动端设备，确保在任何屏幕尺寸下都有良好的浏览体验。
- 加载速度：首页加载速度必须快。需优化图片资源、代码压缩，并合理利用缓存技术。首屏加载时间应控制在2秒以内。
- 浏览器兼容：兼容主流的现代浏览器（如Chrome、Edge、Safari、Firefox的最新两个主要版本）。

---

五、技术需求

- 前端：建议使用 Next.js 或 TanStack Start 作为React框架。UI组件库采用 Ant Design，样式方案使用 Tailwind CSS。状态管理使用 Zustand，数据获取与缓存使用 TanStack Query。
- 后端：采用 Go 语言 + PostgreSQL 数据库。
- SEO友好：利用Next.js的服务端渲染（SSR）或静态站点生成（SSG）能力，确保每个页面都有独立的标题（Title）、描述（Description）和关键词（Keywords），并支持生成站点地图（Sitemap），便于搜索引擎收录。
- 部署：可考虑部署在 Vercel 或 Netlify 等平台，配合 GitHub 进行自动化部署。

六、验收
6.1验收标准（关键示例）
[] 所有页面在Chrome、Safari、Edge最新版上显示正常，功能可用。
[] 在手机和PC上访问，页面布局均能自适应且体验良好。
[] 首页在标准网络环境下的首屏加载时间小于2秒。
[] 后台可以成功新增一个项目，并能在前台"项目介绍"页面正确显示。
[] 后台可以成功更新"毕业去向"信息，并能在前台正确展示。
[] 所有链接（如"加入我们"的报名链接）均能正确跳转。
